## Tentang

Aplikasi untuk ISP lokal

## Run Locally

Clone the project

```bash
  git clone https://github.com/vermaysha/isp-admin.git
```

Go to the project directory

```bash
  cd isp-admin
```

Install php dependencies

```bash
  composer install
```

Start the server

```bash
  ./vendor/sail up
```

Open new terminal & Install npm dependencies
```bash
  ./vendor/sail npm install
```

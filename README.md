
# ISP Admin

Aplikasi untuk ISP lokal

## Run Locally

Clone the project

```bash
  git clone https://github.com/vermaysha/isp-admin.git
```

Go to the project directory

```bash
  cd isp-admin
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  ./vendor/sail up
```
